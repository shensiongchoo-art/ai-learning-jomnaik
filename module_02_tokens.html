Tokens — The Lego Bricks of Language

LLMs don't read words. They read tiny pieces called tokens. Let's see them in action.

The Analogy

Think of a sentence like a Lego structure. You don't build with whole words — you build with bricks. An LLM breaks every sentence into small pieces (tokens) before it can read it. Some tokens are whole words. Some are just parts of words. Some are punctuation.

Why Does This Matter?

🧱 Why tokens matter to you

  • LLMs have a limit on how many tokens they can process at once — called the context window (we'll learn this in Module 3)
  • When you pay to use an LLM API, you pay per token — so longer messages cost more
  • Unusual words (like technical jargon or names) often split into more tokens, which can affect quality
  • Understanding tokens helps you write better prompts — shorter, clearer prompts use fewer tokens and often get better results
