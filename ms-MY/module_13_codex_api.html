Codex dan OpenAI API

Bezakan coding agent untuk repo dengan API untuk membina aplikasi sendiri.

Modul 13

Codex

Codex ialah pengalaman coding agent yang boleh membantu memahami repo, mencadang perubahan, menjalankan arahan, dan menyediakan patch bergantung pada izin serta persekitaran.

OpenAI API

API ialah cara pembangun menghantar permintaan kepada model dari aplikasi sendiri. Aplikasi menghantar input, pilihan model, dan tools yang dibenarkan, kemudian menerima respons.

Keselamatan API key

API key ialah rahsia. Simpan di server, environment variable, atau secret manager. Jangan letak dalam HTML awam, aplikasi frontend, atau GitHub repository.

Aliran kerja baik

Untuk Codex, semak diff dan jalankan ujian. Untuk API, log ralat, kawal kos, hadkan akses, dan reka fallback. Kedua-duanya perlu pemantauan.
