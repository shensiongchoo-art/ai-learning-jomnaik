Ciri ChatGPT

Kenali tools, memory, data analysis, files, dan GPTs sebagai ciri produk yang berubah mengikut akses.

Modul 12

Tools

ChatGPT boleh mempunyai tools seperti search, data analysis, file upload, image input, voice, image generation, atau connected apps. Ketersediaan bergantung pada pelan, workspace, lokasi, dan tetapan.

Memory

Memory boleh membantu ChatGPT mengingati pilihan atau maklumat merentas perbualan jika dihidupkan. Pengguna perlu tahu cara melihat, mengurus, atau mematikan memory.

Data analysis

Untuk fail dan data, ChatGPT boleh membantu membersih, menganalisis, memvisualkan, atau menerangkan data. Semak kod, andaian, dan hasil kerana analisis tetap boleh salah.

GPTs

GPTs ialah pembantu tersuai dalam ChatGPT yang boleh menggabungkan arahan, pengetahuan, dan tools. Ia bukan latihan model baharu penuh; ia lebih kepada konfigurasi pengalaman.
