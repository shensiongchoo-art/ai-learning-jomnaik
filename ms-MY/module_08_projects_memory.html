Projects, Memory, dan RAG

Bezakan ruang kerja, memori produk, dan carian dokumen.

Modul 08

Projects

Projects ialah ruang kerja untuk mengumpulkan perbualan, arahan, dan bahan berkaitan satu topik. Ciri sebenar bergantung pada produk dan pelan.

Memory

Memory ialah ciri produk yang boleh mengingati maklumat merentas perbualan jika tersedia dan dihidupkan. Pengguna perlu faham tetapan, kawalan, dan polisi data sebelum menggunakannya.

RAG

Retrieval-augmented generation (RAG) bermaksud sistem mencari dokumen relevan dahulu, kemudian menggunakan petikan itu sebagai konteks untuk menjawab. Ini membantu jawapan lebih berpijak pada sumber.

Pilih yang sesuai

Gunakan Projects untuk mengurus kerja, Memory untuk pilihan peribadi yang stabil, dan RAG untuk pengetahuan dokumen yang perlu dirujuk semula. Jangan campurkan data sensitif tanpa kebenaran.
