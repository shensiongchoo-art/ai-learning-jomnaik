Projects, Memory, dan RAG

Bezakan ruang projek, memori produk, dan retrieval untuk dokumen.

Cerita

Panel 1 — Projects
📁
Projects
Project ialah ruang kerja khusus untuk satu topik, pelanggan, kursus, atau produk. Ia boleh menyimpan arahan dan fail rujukan.
Konsep
Panel 2 — Memory
📁
Memory
Memory ialah ciri produk yang boleh menyimpan maklumat merentas perbualan jika tersedia dan diaktifkan. Ia perlu diurus dalam tetapan.
Cara fikir
Panel 3 — RAG
📁
RAG
Retrieval-augmented generation (RAG) mengambil maklumat relevan daripada dokumen atau sumber ketika menjawab, bukan semestinya melatih semula model.
Amalan

Ringkasan

Perkara penting

  • Projects membantu konteks jangka panjang.
  • Memory bukan ingatan sempurna.
  • Data sensitif perlu ikut polisi organisasi.
