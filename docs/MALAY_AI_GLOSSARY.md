# Malaysian Malay AI glossary

This glossary is the terminology anchor for the `ms-MY/` pages. It is not a substitute for professional translation review, but it reduces ad hoc AI wording by grounding core terms in DBP/PRPM where available.

## DBP/PRPM-backed terms

| English | Malaysian Malay used here | Source note |
| --- | --- | --- |
| artificial intelligence / AI | kecerdasan buatan / AI | PRPM lists `kecerdasan buatan` for `artificial intelligence (AI)` in Teknologi Maklumat and Kamus Dewan entries for `AI`. |
| large language model / LLM | model bahasa besar (large language model, LLM) | PRPM lists `model bahasa besar (LLM)` in the Kecerdasan Buatan field. |
| context | konteks | Kamus Dewan and Kamus Komputer entries define `konteks`; this is appropriate for model input/background. |
| machine translation | terjemahan mesin | PRPM lists `terjemahan mesin` for `machine translation`. |

## Style rule

Prefer natural Malaysian Malay over literal translation. If an English AI term is what learners will see in real product documentation, keep the English term and explain it clearly in Malay.
