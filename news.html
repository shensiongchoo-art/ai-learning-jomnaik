How this feed now works

A scheduled GitHub Action runs scripts/fetch-ai-news.mjs, fetches public RSS feeds server-side, appends newly discovered items into data/news-feed.json, sorts newest first, and commits the updated JSON back to the repo.

Why this is safer

  • No browser-side CORS proxy dependency for the main feed.
  • Stored JSON gives a historical append-only style feed.
  • The page renders text with DOM APIs, not raw RSS HTML injection.

Manual update

Go to GitHub Actions → Update AI News Feed → Run workflow. If new headlines are found, the action commits them automatically.
